Git Hub Tutorial for Beginners

There are few things of git you should learn as a beginner.

  • Open a folder you want to use git
  • Open Command prompt/Git Bash in that folder

Git Commands

Git Initialization :

git init

--First command to tell IDE to Initialize git fot that folder

Adding File :

git add [file name]

--To add file to stagging area

git add .

--To add all file to stagging area

Checking Status of files :

git status [file name](optional)

--To check the current status of file, it is the most frequent command we use in git

Commit files :

git commit -m [message]

--To move file from stagging area to unmodified area, write message to remember what you have change.

Connect to GitHub Repository :

git remote add origin [git repository address]

--To Connect your local repository(folder) to git hub repository.

Update GitHub Repository :

git push -u origin [branch name]

--To update git hub repository with local repository (by specific branch).
